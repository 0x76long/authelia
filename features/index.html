Features

Authelia is a 2FA & SSO authentication server which is dedicated to the security of applications and users. It can be considered as an extension of reverse proxies by providing features specific to authentication. You will find among other features:

  • Multiple two-factor methods.
  • Identity verification when registering second factor devices.
  • Reset password.
  • Ban account after too many attempts (known as regulation).
